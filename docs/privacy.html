2. What the App Does

Jedum Format Forge lets users convert files on their device and, for selected formats, through our cloud conversion backend.

3. Information We Process

  • Files selected by the user for conversion
  • Conversion settings and output preferences
  • Temporary session identifiers used to secure cloud conversion requests
  • Advertising and consent information when ads are enabled
  • Basic server metadata such as request timestamps, request size, and abuse-prevention logs

We do not intentionally collect special category personal data through the app.

4. Local Processing

Many conversions happen entirely on the device. When a conversion is processed locally, the selected file is not uploaded to our cloud backend for that conversion step.

5. Cloud Conversions

Some conversion types require our backend service, including server-assisted document conversion.

When a cloud conversion is used, the app may upload:

  • the file selected for conversion
  • the requested output format
  • a temporary app session token or technical identifier required to secure the service

Cloud conversion files are processed only to complete the requested conversion.
