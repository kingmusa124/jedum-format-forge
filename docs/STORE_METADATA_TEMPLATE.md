## Full description

Jedum Format Forge is a local-first file conversion app designed for a modern mobile workflow. Convert images, PDFs, Word documents, spreadsheets, and presentation files in a focused workspace built for speed and clarity.

Core highlights:

- Local-first conversion for supported file types
- Cloud-ready workflows through your own backend
- Clean conversion history and result previews
- Save, preview, open, and share converted files
- Modern interface designed for phones and tablets

Use Jedum Format Forge to keep file conversion simple, organized, and reliable whether you are working on a phone or tablet.
